# VenueHR Landing Page

This is the professional landing page for VenueHR, designed to showcase the product's features to wedding venue owners.

## Features Showcased
- **Digital Agreements:** Staff Uniform & Professional Conduct compliance.
- **SMS Onboarding Bot:** Automated data collection from new staff.
- **Staffing Matrix:** Ratio-based headcount calculation.
- **SMS Time Tracking:** Clock-in/out with GPS and break compliance.
- **FAQ Auto-Reply:** Instant answers to common staff questions via SMS.

## Deployment to Netlify
1.  Connect this repository to Netlify.
2.  Set the **Base directory** to `landing_page`.
3.  Set the **Build command** to `none` (it's a static site).
4.  Set the **Publish directory** to `.` (within the `landing_page` folder).
5.  Deploy!

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Inter Font (via Google Fonts)
